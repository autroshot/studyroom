# HTTP

HTTP(HyperText Transfer Protocol)는 HTML 문서와 같은 리소스들을 가져올 수 있도록 해주는 [프로토콜](../00-Terminology/01-용어.md#프로토콜(Protocol))이다. HTTP는 웹에서 이루어지는 모든 데이터 교환의 기초이며, 클라이언트와 서버 간의 프로토콜이기도 하다.
